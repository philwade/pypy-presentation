How are some popular languages implemented?

  • Ruby - C
  • Java - C++
  • Go - C
  • JavaScript (V8) - C++
  • Python - C

Python actually has several implementations.

  • CPython - what most people mean when they say "Python"
  • IronPython - for the mono/.NET platform
  • Jython - written in Java, runs on JVM
  • Unladen Swallow - Optimized CPython, written by Google, discontinued
  • Psyco - an JIT for Python, written in C, discontinued
  • PyPy - Python in Python

C is hard, let's go shopping

  • PyPy is billed as Python in Python
  • An academic project, inspired by Psyco, started in 2003
  • The project has two pieces - RPython Toolchain and PyPy
  • PyPy is the interpreter written in RPython, compiled by the RPython toolchain

RPython

  • So, I sort of lied when I said Python
  • Specific subset of Python
  • Suited for static analysis
  • Variables must keep type
  • Can't create classes at runtime
  • For loops only with built in data structures

What does the toolchain do?

  • The tool chain does a load of analysis
  • Generates an efficient executable for the target platform
  • Currently supported: C/POSIX, .NET, JVM
  • Automagically created JIT within the executable

So, why PyPy?

Speed!

  • PyPy is fast
  • Best performance gains on cpu bound tasks
  • http://speed.pypy.org/
  • Example

Tradeoffs

  • Slower at some things
  • Compatibility

Slower

  • Tests, quick running code
  • JIT needs time to warm up
  • Things CPython implements in pure C
  • RPython re-implementations have some trouble keeping up
  • Big Dicts (maybe)
  • Profile to find out
  • Example

What doesn't it do?

  • Py3k
  • NumPy
  • Python C API
  • A few little coding quirks
